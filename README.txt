В данному репозиторій було реалізовано програму знаходження елементів, у яких сума індексів дорівнює значенню. Було реалізовано 2-ма способами Work Dealing, Work Stealing. На не великих масивах, до максимальної кількості потоків на процесорі, Work Dealing та Work Stealing показують дуже схожу швидкість роботи, але при збільшені масиву, метод Work Stealing стає набагато швидшим. Це пов'язано с тим, що в Work Dealing потоки можуть простоювати, що суттєво збільшує час обрахунку.
